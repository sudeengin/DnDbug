🎯 Key Features Implemented

  • Split Layout: Left pane (w-80) for scenes list, right pane (flex-1) for workspace
  • Scene List: Shows title + objective + status badges with gating logic
  • Scene Workspace: Header with actions + sub-tabs (Detail, Context Out, History)
  • Gating Rules: Cannot access Scene N+1 until Scene N is Locked
  • Status Management: Draft → Generated → Edited → Locked → NeedsRegen
  • Context Integration: effectiveContext from locked scenes only
  • URL Persistence: ?sessionId=...&scene=N parameters

🎮 Workflow Test

  1. Generate Detail: Click "Generate Detail" on Scene 1 → Status: Draft → Generated
  2. Edit Scene: Make changes in Detail tab → Status: Generated → Edited
  3. Lock Scene: Click "Lock Scene" → Status: Edited → Locked
  4. Generate Next: Click "Generate Next" → Navigate to Scene 2
  5. Context Flow: Scene 2 uses effectiveContext from locked Scene 1
  6. Unlock Impact: Unlock Scene 1 → Scene 2+ marked as NeedsRegen
